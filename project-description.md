# Face Signal Lab

GitHub description:
Browser-only face signal visualization demo: camera-based Face Mesh landmarks, head tilt, pose, lighting status and Canvas overlay. Built with JavaScript, WebRTC and local MediaPipe Face Mesh; no backend or external API.

Короткое описание:
Браузерный эксперимент с камерой для визуализации лицевой геометрии: landmarks, положение лица, наклон головы, освещение и self-reported profile.

Описание для портфолио:
Face Signal Lab - self-contained frontend demo для анализа визуальных сигналов лица прямо в браузере. Приложение использует WebRTC-камеру и MediaPipe Face Mesh, строит Canvas-overlay с лицевыми ориентирами, рамкой лица, положением, наклоном, статусом освещения и простыми expression cues. Возраст, пол и настроение не определяются автоматически: это поля, которые пользователь заполняет сам. Проект не использует backend, аккаунты, ключи или внешние API; runtime-файлы MediaPipe Face Mesh и Lucide подключены локально.

Стек:
HTML, CSS, JavaScript, Canvas API, WebRTC/getUserMedia, MediaPipe Face Mesh, Lucide.
